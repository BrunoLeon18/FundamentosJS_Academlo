Ejercicio #1

  • Agrega un div con una clase container
  • Dentro agrega tres párrafos
  • Cada párrafo debe tener una clase
  • Las clases deben ser:
    1. parrafo1
    2. parrafo2
    3. parrafo3
  • El parrafo1 debe tener la primera letra en mayúscula y un tamaño de 30px con la familia de fuente 'Times New Roman', Times, serif
  • El parrafo2 debe tener un color de fondo lightblue y un color de letra black
  • El parrafo3 el texto debe ser centrado y subrayado
